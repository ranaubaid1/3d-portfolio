# Ubaid Abid - Interactive 3D Developer Portfolio

A state-of-the-art, fully interactive 3D WebGL developer portfolio built using **Three.js**, **JavaScript**, and premium **CSS Glassmorphism** layouts.

## 🚀 Live Demo & Visual Highlights

- **3D Robot Companion**: A floating 3D model that follows pointer movements (mouse or touchscreen) and reacts to clicks with gravity-based jumps and velocity spins.
- **Orbiting Specular Light Spotlights**: Dual PointLights (cyan & magenta) orbiting the robot model to cast shifting colored reflections across its metallic body.
- **3D Glass Card Tilt Effect**: Glassmorphic panels containing biography details and project lists that pitch and tilt dynamically in 3D space relative to cursor positions.
- **Voice-Over Text-to-Speech Autoplay**: Features native browser voice synthesis that automatically reads out card details when navigating between sections.
- **WebGL Starfield Galaxy**: 2,000 orbiting particles swirling slowly in the background space.
- **Responsive Layout**: Fluid CSS typography (`clamp`) and flexible columns scaling down to support mobile touch devices seamlessly.

---

## 🛠️ Tech Stack

- **Graphics & 3D Rendering**: Three.js (WebGL)
- **Frontend & Styling**: Vanilla HTML5, CSS3, JavaScript (ES6 Modules)
- **Typing Animations**: Typed effect scripts
- **Local Dev Server**: Node.js & Express.js

## 💻 Local Setup & Development

Follow these steps to run the portfolio locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ranaubaid1/3d-portfolio.git
   cd 3d-portfolio
   ```

2. **Install Node dependencies:**
   ```bash
   npm install
   ```

3. **Start the local Express server:**
   ```bash
   node server.js
   ```

4. **Open in Browser:**
   Go to [http://localhost:3000](http://localhost:3000) to view the live interactive page.
